SKBEL
=================================

.. currentmodule:: SKBEL

This package implements the Bayesian Evidential Learning framework.

Installation
------------

SKBEL is available through `PyPI <https://pypi.org/project/skbel/>`_, and may be installed using ``pip``: ::

   $ pip install skbel
